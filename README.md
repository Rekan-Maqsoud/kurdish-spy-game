# سیخوڕ - یاری کوردی 🕵️

یاری سیخوڕ بۆ مۆبایلی ئەندرۆید بە زمانی کوردی

## 📱 دەربارەی یاری

سیخوڕ یاریەکی کۆمەڵایەتییە کە چەند کەس دەیکەن. یەکێکیان سیخوڕە و وشەکە نازانێت، و دەبێت خۆی بشارێتەوە لە یاریزانەکانی دیکە.

## ✨ تایبەتمەندییەکان

- 🇮🇶 بە زمانی کوردی (سۆرانی)
- 📝 زیاتر لە 1000 وشە
- 📂 20 پۆلی جۆراوجۆر (خواردن، ئاژەڵ، شوێن، و هتد)
- 🎨 دیزاینی مۆدێرن Glass UI
- 💾 پاشەکەوتی ناوخۆیی (Local Storage)
- 🏆 خشتەی خاڵە باڵاکان
- ⚙️ ڕێکخستنی یاری (ژمارەی گەڕ، خاڵەکان، پۆلەکان)

## 🎮 چۆنیەتی یاریکردن

1. **یاریزانەکان دیاری بکە** - لانیکەم 3 یاریزان
2. **وشەکان دابەش بکە** - هەر یاریزانێک وشەکەی خۆی دەبینێت (یەکێکیان سیخوڕە)
3. **گفتوگۆ** - پرسیار بکەن بەبێ گوتنی وشەکە
4. **دەنگدان** - دەنگ بدەن بۆ سیخوڕ
5. **تەخمینی سیخوڕ** - سیخوڕ هەوڵدەدات وشەکە بدۆزێتەوە

## 🏅 سیستەمی خاڵەکان

- **+1 خاڵ** - بۆ دۆزینەوەی سیخوڕ
- **+2 خاڵ** - بۆ سیخوڕ ئەگەر وشەکەی دۆزیەوە

## 🚀 چۆن جێبەجێی بکەیت

```bash
# کلۆن بکە
git clone https://github.com/Rekan-Maqsoud/kurdish-spy-game.git
cd kurdish-spy-game

# پاکەجەکان دامەزرێنە
npm install

# بۆ development
npx expo start

# بۆ ئەندرۆید
npx expo start --android

# بۆ وێب
npx expo start --web
```

## 🌐 بڵاوکردنەوە بۆ وێب

```bash
# Export بکە بۆ وێب
npx expo export --platform web

# فایلەکان دەبنە dist/ فۆڵدەر
# ئەتوانیت بۆ Vercel یان Netlify یان GitHub Pages ئەپڵۆد بکەیت
```

## 📦 بۆ Play Store ئامادە بکە

```bash
# EAS CLI دامەزرێنە
npm install -g eas-cli

# لۆگین بکە
eas login

# بیلد بکە بۆ ئەندرۆید
eas build --platform android
```
